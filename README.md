# 🎵 Hand Gesture Controlled Music Player (Sa Re Ga Ma)

A web-based project that allows users to play **Indian classical musical notes (Sa Re Ga Ma Pa Dha Ni Sa)** using **hand gestures through a webcam**. The system detects hand movements and triggers the corresponding musical note in real time.

This project demonstrates how **computer vision and web technologies** can be combined to create an interactive gesture-based musical instrument.

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript** – Application logic
* **Hand Tracking Library (MediaPipe / TensorFlow.js)** – Gesture recognition
* **Web Audio API** – Playing musical notes

---

## ⚙️ How It Works

1. The browser requests access to the **webcam**.
2. A **hand tracking model** detects the position of the hand and fingers.
3. Each gesture corresponds to a **specific musical note**.
4. When a gesture is detected, the system plays the corresponding **Sa, Re, Ga, Ma, Pa, Dha, Ni, Sa / Do, Re, Mi, Fa, So, La, Ti, Do** note.
